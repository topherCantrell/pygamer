## Specs

  - ATSAMD51 with 512K flash and 192K RAM. 8M of QSPI flash.
  - 160x128 (18 bit color) TFT display with backlight (SPI)
  - dual-potentiometer analog stick
  - 4 buttons
  - 5 NeoPixels
  - Light sensor
  - 3 axis accelerometer
  - Mono sound
